# Typing Game

A fun and interactive typing speed game built with React. The game measures how fast you can type!

---

## Features

- Focus automatically on the text area using `useRef`
- Timer-based challenge
- Word counter and accuracy tracker
- Responsive design

---

## Technologies Used

- React.js
- HTML & CSS
- JavaScript (ES6)

## Installation

1. Clone the repository:

   ```bash
   git clone "repo directory"

   ```

2. Go to the project folder:

   ```bash
   cd "typing-game"

   ```

3. Use `npm install` to install dependencies.

4. Use `npm start` to run the project.
